Splay Tree

Title: Splay Tree
Author: Tobias Nipkow
Submission date: 2014-08-12
Abstract: Splay trees are self-adjusting binary search trees which were invented by Sleator and Tarjan [JACM 1985]. This entry provides executable and verified functional splay trees.

The amortized complexity of splay trees is analyzed in the AFP entry Amortized Complexity.
